projet_kbd.data_downloader module

Data Downloader Module

This module handles the automated download of required dataset files from Google Drive if they are not already present in the specified data directory.

The DATA_FILES dictionary maps filenames to their respective Google Drive links. The function download_data checks if each file exists in the provided directory, and if not, downloads it using the gdown library.
