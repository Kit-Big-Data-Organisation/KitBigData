projet_kbd.data_loader moduleÔÉÅ

DataLoader ModuleÔÉÅ

This module provides functionality for loading, preprocessing, and enhancing recipe data. It includes methods for merging datasets, adding derived features (such as year and cuisines), and preparing data for analysis.
